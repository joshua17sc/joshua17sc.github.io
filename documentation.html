Introduction

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

Elements

HTML elements are the building blocks of HTML pages.

    Types of elements in HTML
  • div (Dividers)
  • p (Paragraphs)
  • a (Anchors)
  • nav (Navigation)
  • img (Images)

Classes

The class attribute provides a way of classifying similar elements.

This can be used for semantic or presentation purposes.

IDs

The id attribute provides a document-wide unique identifier for an element.

his is used to identify the element so that stylesheets can alter its presentational properties, and scripts may alter, animate or delete its contents or presentation.

Divs

a div tag breaks up a page with semantically unspecific divisions

div tags are used to break down sections of the page in ways that do not necessarily fit a semantic tag page element
